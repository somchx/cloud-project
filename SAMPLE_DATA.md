# ข้อมูลตัวอย่างที่มีใน Database

เมื่อรันคำสั่ง `python seed_data.py` ระบบจะสร้างข้อมูลตัวอย่างดังนี้:

## หมวดหมู่สินค้า (Categories)

1. **ขนม** - ขนมและของว่าง
2. **เครื่องดื่ม** - น้ำดื่มและเครื่องดื่มต่างๆ
3. **ของใช้** - ของใช้ในบ้าน
4. **อาหารแห้ง** - อาหารแห้งและของกระป๋อง

## สินค้า (Products)

### หมวดหมู่: ขนม
- เลย์ รสออริจินัล - ฿20 (คงเหลือ 50 ชิ้น)
- โดริโทส - ฿25 (คงเหลือ 30 ชิ้น)
- ช็อคโกแลต - ฿15 (คงเหลือ 40 ชิ้น)

### หมวดหมู่: เครื่องดื่ม
- น้ำดื่ม 600ml - ฿7 (คงเหลือ 100 ชิ้น)
- โค้ก 325ml - ฿13 (คงเหลือ 60 ชิ้น)
- เป็ปซี่ 325ml - ฿13 (คงเหลือ 60 ชิ้น)
- น้ำชาเขียว - ฿15 (คงเหลือ 45 ชิ้น)

### หมวดหมู่: ของใช้
- ทิชชู่ - ฿30 (คงเหลือ 25 ชิ้น)
- ถุงขยะ - ฿35 (คงเหลือ 20 ชิ้น)
- สบู่ - ฿25 (คงเหลือ 30 ชิ้น)

### หมวดหมู่: อาหารแห้ง
- มาม่า - ฿7 (คงเหลือ 80 ชิ้น)
- ยำยำ - ฿7 (คงเหลือ 70 ชิ้น)

## วิธีเพิ่มข้อมูลตัวอย่างเพิ่มเติม

แก้ไขไฟล์ `backend/seed_data.py`:

```python
# เพิ่มหมวดหมู่ใหม่
categories_data.append({
    "name": "เครื่องสำอาง",
    "description": "เครื่องสำอางและความงาม"
})

# เพิ่มสินค้าใหม่
products_data.append({
    "name": "ยาสีฟัน",
    "price": 45,
    "category_id": 3,
    "stock": 30
})
```

จากนั้นรัน:
```bash
cd backend
python seed_data.py
```
